# Cloud Administration Scripts

This directory groups cloud admin scripts by platform.

## Subfolders

- [Azure](./Azure/) - Azure resource management and reporting scripts
- [Microsoft 365](./Microsoft365/) - Microsoft 365 and Exchange Online administration

## Usage

```powershell
# Example: list Azure VMs
cd .\scripts\Cloud\Azure
.\Manage-AzureVMs.ps1 -Action List
```
